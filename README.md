## 谷粒商城电商项目

### 使用 vue+vuex-vue-router+element开发

### 接口使用的是尚硅谷提供的公共接口
http://gmall-h5-api.atguigu.cn/api

#### 使用
##### 安装
`git clone git@github.com:0Shino0/20220421_gshop-client.git`
##### 下载依赖
```yaml
    npm i

    # 或者
    yarn
```


##### 展示
`yarn serve`
